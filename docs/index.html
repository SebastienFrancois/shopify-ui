Installation

Choose the method that works best for your Shopify store.

Option 1: CDN (Recommended)

The fastest way to get started. Add one line to your theme.

1

Open your Shopify Admin and go to Online Store → Themes → Edit Code

2

Open theme.liquid and paste before </head>:

<script type="module"
  src="https://unpkg.com/@shopify-ui/components/dist/shopify-ui.es.js">
</script>
3

Use any component in your Liquid templates. Done!

Option 2: Theme Assets

Self-host the library in your Shopify theme for full control.

1

Download shopify-ui.es.js from the latest release.

2

Upload it to your theme's Assets folder.

3

Add to theme.liquid:

<script type="module"
  src="{{ 'shopify-ui.es.js' | asset_url }}">
</script>

Option 3: npm

For custom Shopify themes with a build step or headless setups.

1

Install the package:

npm install @shopify-ui/components
2

Import in your JavaScript:

// Import everything
import '@shopify-ui/components';

// Or tree-shake individual components
import '@shopify-ui/components/product-card';

Liquid Snippets (Optional)

Pre-built Liquid snippets for seamless Shopify integration.

1

Copy the .liquid files from shopify/snippets/ into your theme's snippets/ folder.

2

Use them with Liquid's render tag:

{% render 'ecom-product-card', product: product %}
{% render 'ecom-carousel', images: product.images %}

Customization

All components use CSS Custom Properties. Override them in your theme to match your brand.

/* Add to your theme.css or theme.liquid */
:root {
  --sui-primary: #2563eb;       /* Brand color */
  --sui-primary-hover: #1d4ed8;  /* Hover state */
  --sui-border-radius: 8px;     /* Corner rounding */
  --sui-font-family: 'Your Font', sans-serif;
  --sui-shadow: 0 4px 6px -1px rgba(0,0,0,0.1);
}

Events

// Listen for component events
document.addEventListener('ecom:add-to-cart', (e) => {
  console.log('Added to cart:', e.detail);
});

document.addEventListener('ecom:slide-change', (e) => {
  console.log('Slide index:', e.detail.index);
});

document.addEventListener('ecom:chat-message', (e) => {
  console.log('Chat message:', e.detail);
});

document.addEventListener('ecom:bundle-add', (e) => {
  console.log('Bundle added:', e.detail);
});
